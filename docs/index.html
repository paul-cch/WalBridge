WalBridge GitHub

Linux-inspired theming for macOS

Bridge your wallpaper to your entire desktop stack.

WalBridge automatically syncs color themes across 14 apps from your current wallpaper, with high-contrast active borders, dimmed inactive borders, and animated transitions.

Get Latest Release View Source

Quick Install

git clone https://github.com/paul-cch/WalBridge.git
cd WalBridge
bash install.sh
# optional setup for tmux/btop/iTerm2
bash install.sh --setup-targets

Supported Targets

SketchyBarJankyBordersKittyWezTermAlacrittyGhosttyiTerm2tmuxbtopNeovimYaziStarshipOpenCodeHydroToDo

How It Works

  1. Capture current wallpaper per display (with fallback for dynamic wallpapers).
  2. Extract dominant palette and generate a coherent scheme.
  3. Write target configs atomically, reload supported apps, and skip unchanged wallpapers via perceptual hash.
